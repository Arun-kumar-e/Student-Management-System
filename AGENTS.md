# Student Management System — Frontend

## Stack
- React + Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- React Router
- Vitest + React Testing Library

## Skills to load
Load the following skills from `.agents/skills/` when working on this project:

- `react-vite-best-practices` — React + Vite scaffolding and conventions
- `tailwindcss-advanced-design-systems` — Tailwind CSS utility classes and design systems
- `framer-motion-animator` — Framer Motion transitions and gesture code
- `openapi-to-typescript` — Typed request/response from Spring REST endpoints
- `vitest-testing` — Vitest + React Testing Library patterns
- `frontend-design` — Design system thinking and UI/UX
- `figma-implement-design` — Translating designs to code
- `java-springboot` — Backend CORS config and dependency updates

## Backend API
The backend is a Spring Boot app in the same repo. All REST endpoints are under `src/main/java/`.
